Ventana de Avance
=================
Es un simple extension template, que tiene por objetivo agregar una ventana de avance, que se abra, incremente, y cierre, a voluntad del usuario.
Es muy util cuando se necesita realizar algún proceso que pueda demorar, y se necesite mostrarle algo al usuario.

Utilización
===========
- Insertar el template como extension Template
- Configuar el texto de los carteles, y cada cuantas llamadas a la rutina de avance se incrementa la barra
- En el código, agregar 
Para abrir la ventana:
	DO AbrirVentanaEsperaAvance

Para Cerrar la ventana:
	DO CerrarVentanaEsperaAvance

Para Avanzar:
	DO AvanceEspera
